# Install Odoo via the `.deb` package

## 1. بروزرسانی سیستم
sudo apt update && sudo apt upgrade -y

## 2.📦 نصب پیش‌نیازها
sudo apt install python3-pip build-essential python3-dev libpq-dev \
libxml2-dev libxslt1-dev zlib1g-dev libsasl2-dev libldap2-dev \
libjpeg-dev libffi-dev libtiff5-dev libopenjp2-7 liblcms2-dev \
libwebp-dev libharfbuzz-dev libfribidi-dev libxcb1-dev nodejs npm -y


## 3. 🐘 نصب پایگاه داده PostgreSQL
sudo apt install postgresql postgresql-contrib -y
sudo -u postgres createuser -s odoo

## 4. 📄 نصب wkhtmltopdf برای تولید PDF
sudo apt install wkhtmltopdf -y

## 5. 📥 نصب فایل `.deb` اودو

اگر فایل `odoo_18.0+e.latest_all_odoers_ir.deb` را در پوشه Downloads دارید:
cd ~/Downloads
sudo dpkg -i odoo_18.0+e.latest_all_odoers_ir.deb
sudo apt --fix-broken install -y

# بعد از نصب

## 1. بررسی سرویس Odoo
sudo systemctl status odoo

اگر وضعیتش `active (running)` بود، یعنی همه‌چیز خوبه. اگر نبود، با دستور زیر فعالش کن:

sudo systemctl start odoo

و برای اینکه بعد از هر ری‌استارت سیستم خودش اجرا بشه:
sudo systemctl enable odoo

## 2. باز کردن Odoo در مرورگر 🌐
حالا مرورگر رو باز کن و این آدرس رو وارد کن:
http://localhost:8069
اگر روی سرور نصب کردی و می‌خوای از سیستم دیگه وصل بشی، باید IP سرور رو جایگزین `localhost` کنی.

## 3. ساخت دیتابیس و ورود به محیط Odoo

اگر همه‌چیز درست باشه، صفحه‌ای ظاهر می‌شه که ازت می‌خواد دیتابیس بسازی. اطلاعات زیر رو وارد کن:

- **Master Password**: معمولاً `admin` هست، مگر اینکه موقع نصب تغییر داده باشی.
    
- **Database Name**: هر اسمی که دوست داری.
    
- **Email / Login**: ایمیل ادمین.
    
- **Password**: رمز عبور برای ورود.
    
- **Language**: فارسی یا انگلیسی.
    
- **Country**: ایران یا هر کشور دیگه.
    

روی دکمه "Create Database" کلیک کن و وارد محیط Odoo شو 🎉
