 **🚀 راهنمای جامع نصب Odoo Enterprise با Python 3.12 و فایل `.deb`**

این راهنما مراحل کامل نصب و راه‌اندازی Odoo Enterprise را روی سیستم لینوکس (Ubuntu/Debian) با استفاده از فایل `.deb` و محیط مجازی Python 3.12 پوشش می‌دهد.

---

## 🧱 مرحله ۱: نصب پیش‌نیازهای سیستمی

```bash
sudo apt update
sudo apt install -y python3.12 python3.12-venv python3.12-dev \
    build-essential libldap2-dev libsasl2-dev libssl-dev \
    libxml2-dev libxslt1-dev zlib1g-dev libjpeg-dev \
    libpq-dev gcc wkhtmltopdf
